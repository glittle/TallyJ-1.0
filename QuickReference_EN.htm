Quick Reference

For more complete instructions, please read the Instructions Page.

  1. Setting up TallyJ for an Election
    1. Load the Election and Community files. (Files must be in the Data folder in the TallyJ folder.)
    2. Configure Election details on the Election page. If more than one computer is being used, ensure that each has a unique Election file with a different Code on the Election page.
  2. Entering Ballots
    1. Enter each ballot as closely as possible to what is on paper, even recording such things as which names are not readable. Type to search for a name, when found, press Enter.
    2. Record the ballot Id on the paper ballot when it is being entered
    3. If more than one computer is being used, the Election files can be merged into one "master" Election file for reporting.
  3. Reporting Election Results
    1. View and print the "Election Summary" report to provide official results of the election. This also displays tie-votes.
