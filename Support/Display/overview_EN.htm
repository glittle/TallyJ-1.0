TallyJ - A Bahá'í Election Tally System

TallyJ facilitates the recording of ballots and votes at a Bahá'í election. It is designed for use at the location of the election, with the completed paper ballots in hand. All names on the ballots are typed into one or more computers and on-screen reports provide instant results. Your copy of TallyJ is currently at version . An updated version may be available on the TallyJ Support Website (see link below).

Getting Started (Chief Teller) (for details, see Instructions)

  • (A printed copy of the Instructions should be on hand at the election site.)
  • Prepare the Community file. This can be created using external tools. Fill with all names if possible.
  • Click "Files" and load the Community file and create a new Election files
  • Click "Election" and fill in all details (and save)
  • Copy the Election and Community files to common location or to each teller's computer. (Make a seperate copy of the Election file for each computer used by Tellers.

Getting Started (Tellers)

  • (First time only)
    • Click "Files" and load the Community file and the Election file for this computer as provided by Chief Teller
    • Click "Election" and ensure that "Code for this computer" is different on each computer
  • Click "Ballots" and enter ballots!
